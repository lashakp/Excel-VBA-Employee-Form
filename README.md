# Excel VBA Employee Management System

## Overview
A fully automated Excel-based Employee Data Entry Form integrated with a secure database. Automates validation, CRUD operations, and ensures accurate and protected data management.

## Features
- Interactive employee data entry form
- Dropdown validation for Gender and Department
- Automated Save, Modify, Delete, and Reset functions
- Database storage with username and timestamp tracking
- User-friendly messages for all actions
- Protected sheets with controlled editable cells
- Robust error handling

## Setup
1. Enable macros in Excel.
2. Open the workbook.
3. Enter data in unlocked fields on the Form sheet.
4. Use command buttons for operations:
   - Save: Save new or modified record.
   - Modify: Edit an existing record.
   - Delete: Remove a record with confirmation.
   - Reset: Clear the form.
